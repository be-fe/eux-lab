# 事情的起因射这样的

某天, 我在一次bug中发现有个顽固的bug, 表格提交总会报错. 经过排查, 尼玛就是俩数值死活不相等. 这时, 记起出道时师父循循的教导:
"孩子, js的float运算是个坑. 切记, 切记."

立马在 Chrome 中尝试错误的例子, 果不其然, 这种酸爽...

```
> 67.01 + 1
68.01

> 670.01 + 10
680.01

> 67000.01 + 1000
68000.01

> 67000000.01 + 1000000
68000000.00999999

```

What the f**k is `68000000.00999999` ???

再来一个

```
> 1.1 + 1.1 + 1.1 + 0 + 4
7.300000000000001
```
