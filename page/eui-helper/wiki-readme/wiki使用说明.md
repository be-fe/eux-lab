# wiki 侧边栏

wiki 侧边栏是wiki中所有内容的关健导航方式. 其中包括有基础的使用方式, 以及利用筛选的手段来定位到感兴趣的内容.

## 基础使用

### 展开与收缩

阅读时, 具有子内容的页面, 可以很容易的进行收缩和展开, 从而保持一个更加清洁的导航视图.

通过点击这些三角符号, 则能收缩展开.

![](twix.png)

同时, 我们还记忆上一次操作的收缩展开状态, 在下一次重载页面时再次读出上次状态.
